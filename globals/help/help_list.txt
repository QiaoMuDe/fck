fck list - 文件和目录列表工具

功能：列出指定目录中的文件和目录，并支持多种排序和过滤选项。

用法: fck list [选项] [路径]

选项：
  -h             打印帮助信息并退出
  -a             列出所有文件和目录，包括隐藏项
  -c             启用颜色输出
  -t             按修改时间排序
  -s             按文件大小排序
  -n             按文件名排序
  -D             列出目录本身，而不是其内容
  -f             只列出文件，不列出目录
  -d             只列出目录，不列出文件
  -L             只列出软链接，不列出其他类型的文件
  -ro            只列出只读文件
  -ho            只列出隐藏文件或目录
  -l             使用长格式显示文件信息，包括权限、所有者、大小等
  -r             反向排序
  -q             在输出时用双引号包裹条目
  -R             递归列出目录及其子目录的内容
  -u             显示文件的用户和组信息
  -ts  <string>  指定表格样式

表格样式，支持以下选项：
  [基础样式]
    none    - 禁用表格样式
    default - 默认样式
    l       - 浅色样式
    r       - 圆角样式
    bd      - 粗体样式
    db      - 双线样式
  
  [彩色样式]  
    cb      - 亮色彩色样式
    cd      - 暗色彩色样式
  
  [背景色样式]
    cbb     - 黑色背景蓝色字体
    cbc     - 青色背景蓝色字体  
    cbg     - 绿色背景蓝色字体
    cbm     - 紫色背景蓝色字体
    cby     - 黄色背景蓝色字体
    cbr     - 红色背景蓝色字体
  
  [前景色样式]
    cwb     - 蓝色背景白色字体
    ccw     - 青色背景白色字体
    cgw     - 绿色背景白色字体
    cmw     - 紫色背景白色字体
    crw     - 红色背景白色字体
    cyw     - 黄色背景白色字体

示例：
  1. 列出当前目录的所有文件和目录，包括隐藏项
     fck list -a
  2. 按修改时间排序列出当前目录的文件和目录
     fck list -t
  3. 只列出当前目录的文件，并按文件大小排序
     fck list -f -s
  4. 递归列出指定目录的内容，并使用长格式显示
     fck list -R -l /path/to/directory
  5. 列出当前目录的只读文件
     fck list -ro
  6. 用白色主题显示文件列表
     fck list -l -tc light
  7. 用黑色主题显示文件列表
     fck list -l -tc dark

注意：
1. 如果不指定路径，默认为当前目录。
2. 排序选项（-t, -s, -n）不能同时使用，后指定的选项会覆盖前一个。
3. 只能选择一种文件类型过滤（-f, -d, -L）。
