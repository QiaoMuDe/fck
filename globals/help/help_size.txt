fck size - 文件或目录大小计算工具

功能：计算指定文件或目录的大小，并以人类可读格式(B/KB/MB/GB/TB)显示

用法：fck size [选项] [路径]

选项：
  [基本选项]
  -h      打印帮助信息并退出
  -c      启用颜色输出
  -H      包含隐藏文件或目录进行大小计算，默认过滤

  [性能选项]
  -j <int> 指定并发数量，默认为-1表示根据CPU核心数自动设置

  [显示选项]
  -ts <string> 指定表格样式

使用说明：
  1. 支持计算单个文件、目录或通配符匹配的文件大小
  2. 对于目录，会递归计算其下所有文件和子目录的总大小
  3. 结果会以对齐的表格形式显示，包含大小和路径两列

表格样式，支持以下选项：
  [基础样式]
    none    - 禁用边框样式
    default - 默认样式
    l       - 浅色样式
    r       - 圆角样式
    bd      - 粗体样式
    db      - 双线样式
  
  [彩色样式]  
    cb      - 亮色彩色样式
    cd      - 暗色彩色样式
  
  [背景色样式]
    cbb     - 黑色背景蓝色字体
    cbc     - 青色背景蓝色字体  
    cbg     - 绿色背景蓝色字体
    cbm     - 紫色背景蓝色字体
    cby     - 黄色背景蓝色字体
    cbr     - 红色背景蓝色字体
  
  [前景色样式]
    cwb     - 蓝色背景白色字体
    ccw     - 青色背景白色字体
    cgw     - 绿色背景白色字体
    cmw     - 紫色背景白色字体
    crw     - 红色背景白色字体
    cyw     - 黄色背景白色字体

示例：
  1. 计算单个文件大小：
     fck size /path/to/file.txt

  2. 计算目录总大小：
     fck size /path/to/directory

  3. 使用通配符计算多个文件大小：
     fck size /path/to/*.txt

  4. 计算当前目录下所有文件大小：
     fck size .

输出格式：
  <大小>          <路径>
  12.5MB         /path/to/file.txt
  1.2GB          /path/to/large_directory

注意事项：
  1. 大小单位会自动选择最合适的(B/KB/MB/GB/TB)
  2. 对于大目录，计算可能需要一些时间
  3. 支持Windows和Unix风格的路径
